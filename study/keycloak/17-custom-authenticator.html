м»ӨмҠӨн…Җ Authenticator

н•ҷмҠө лӘ©н‘ң

  • AuthenticatorмҷҖ AuthenticatorFactory мқён„°нҺҳмқҙмҠӨмқҳ кө¬мЎ°мҷҖ мұ…мһ„мқ„ мқҙн•ҙн•ңлӢӨ.
  • AuthenticationFlowContextмқҳ success / failureChallenge / attempted лҸҷмһ‘мқ„ кө¬л¶„н•ҙ мҳ¬л°”лҘё л¶„кё° мӢңм җмқ„ кі лҘёлӢӨ.
  • FreeMarker н…ңн”ҢлҰҝмңјлЎң мӮ¬мҡ©мһҗ нҸјмқ„ л ҢлҚ”л§Ғн•ҳкі  м ңм¶ң к°’мқ„ кІҖмҰқн•ҳлҠ” л°©лІ•мқ„ мқөнһҢлӢӨ.
  • Authentication Flowм—җм„ң м»ӨмҠӨн…Җ лӢЁкі„лҘј Required/AlternativeлЎң л°°м№ҳн•ҳлҠ” м „лһөмқ„ м„Өкі„н•  мҲҳ мһҲлӢӨ.

1. AuthenticatorмҷҖ AuthenticatorFactory

Keycloakмқҳ лЎңк·ёмқё н”ҢлЎңмҡ°лҠ” мһ‘мқҖ лӢЁмң„мқҳ AuthenticatorлҘј мҲңм„ңлҢҖлЎң мӢӨн–үн•ҳлҠ” мғҒнғң кё°кі„лӢӨ. кё°ліё м ңкіөлҗҳлҠ” "Username Password Form", "OTP Form", "Cookie", "Identity Provider Redirector"к°Җ лӘЁл‘җ к°ҷмқҖ SPIмқҳ кө¬нҳ„мІҙлӢӨ. м»ӨмҠӨн…Җ лӢЁкі„лҘј м¶”к°Җн•ңлӢӨлҠ” кІғмқҖ кі§ AuthenticatorлҘј мғҲлЎң кө¬нҳ„н•ңлӢӨлҠ” лң»мқҙлӢӨ.

мқён„°нҺҳмқҙмҠӨ мҢҚ

мқён„°нҺҳмқҙмҠӨмұ…мһ„
AuthenticatorFactoryProvider ID, н‘ңмӢң мқҙлҰ„, кө¬м„ұ мҠӨнӮӨл§Ҳ, мӢӨн–ү к°ҖлҠҘн•ң Requirement мҳөм…ҳмқ„ м„ м–ё
AuthenticatorмӢӨм ң мқёмҰқ лЎңм§Ғ. authenticate()(мҙҲкё° м§„мһ…)мҷҖ action()(нҸј м ңм¶ң нӣ„ мІҳлҰ¬)мқ„ кө¬нҳ„

FactoryлҠ” м„ңлІ„лӢ№ мӢұкёҖнҶӨмқҙлҜҖлЎң мҷёл¶Җ API нҒҙлқјмқҙм–ёнҠёВ·м»Өл„Ҙм…ҳ н’ҖВ·м„Өм • мәҗмӢңлҘј ліҙкҙҖн•ңлӢӨ. AuthenticatorлҠ” мҡ”мІӯл§ҲлӢӨ мғҲлЎң л§Ңл“Өм–ҙм ё Factoryк°Җ мҙҲкё°нҷ”н•ҙ л‘” мһҗмӣҗмқ„ м“ҙлӢӨ. CH16м—җм„ң м •лҰ¬н•ң Factory/Provider кҙҖкі„к°Җ к·ёлҢҖлЎң м Ғмҡ©лҗңлӢӨ.

Requirement мҳөм…ҳ

Factoryк°Җ л°ҳнҷҳн•ҳлҠ” AuthenticationExecutionModel.Requirement[]лҠ” Admin Consoleм—җм„ң м„ нғқ к°ҖлҠҘн•ң л°°м№ҳ лӘЁл“ңлҘј м ңн•ңн•ңлӢӨ.

RequirementмқҳлҜё
REQUIREDл°ҳл“ңмӢң м„ұкіөн•ҙм•ј н•Ё. мӢӨнҢЁ мӢң лЎңк·ёмқё м „мІҙ мӢӨнҢЁ
ALTERNATIVEк°ҷмқҖ л ҲлІЁмқҳ лӢӨлҘё Alternative мӨ‘ н•ҳлӮҳл§Ң м„ұкіөн•ҳл©ҙ лҗЁ
DISABLEDкұҙл„ҲлңҖ
CONDITIONALмЎ°кұҙ л¶„кё° м„ңлёҢн”ҢлЎңмҡ° лӮҙм—җм„ңл§Ң мӮ¬мҡ©

мӮ¬лІҲ кІҖмҰқмқҖ "кјӯ нҶөкіјн•ҙм•ј н•ҳлҠ” лӢЁкі„"лқј REQUIREDл§Ң мқҳлҜё мһҲлӢӨ. л°ҳл©ҙ PasskeyмҷҖ OTPлҘј лҸҷмӢңм—җ н—Ҳмҡ©н•ҳлҠ” 2м°Ё мқёмҰқмқҙлқјл©ҙ ALTERNATIVEк°Җ н•„мҡ”н•ҳлӢӨ.

2. AuthenticationFlowContext вҖ” л¶„кё° мӢңм җ

Authenticatorмқҳ лӘЁл“  лЎңм§ҒмқҖ AuthenticationFlowContextлҘј нҶөн•ҙ кІ°кіјлҘј ліҙкі н•ңлӢӨ. м–ҙл–Ө л©”м„ңл“ңлҘј нҳём¶ңн•ҳлҠҗлғҗм—җ л”°лқј н”ҢлЎңмҡ° мӢӨн–үкё°к°Җ лӢӨмқҢ лҸҷмһ‘мқ„ кІ°м •н•ңлӢӨ.

stateDiagram-v2
    [*] --> authenticate
    authenticate --> Success : context.success()
    authenticate --> Challenge : context.challenge(form)
    authenticate --> Attempted : context.attempted()
    authenticate --> Failure : context.failure(...)
    Challenge --> action : мӮ¬мҡ©мһҗ нҸј м ңм¶ң
    action --> Success : context.success()
    action --> FailChallenge : context.failureChallenge(...)
    action --> Failure : context.failure(...)
    FailChallenge --> action : мһ¬мӢңлҸ„
    Success --> [*]
    Failure --> [*]
    Attempted --> [*] : лӢӨмқҢ AlternativeлЎң мқҙлҸҷ

мЈјмҡ” л©”м„ңл“ңмҷҖ мҡ©лҸ„

л©”м„ңл“ңмқҳлҜёмӮ¬мҡ© мӢңм җ
success()мқҙ лӢЁкі„ нҶөкіј вҶ’ лӢӨмқҢ мӢӨн–үкё°лЎңм •мғҒ кІҖмҰқ мҷ„лЈҢ
challenge(Response)нҸјмқҙлӮҳ лҰ¬лӢӨмқҙл үнҠё мқ‘лӢөмқ„ лӮҙл Ө мӮ¬мҡ©мһҗ мһ…л Ҙ лҢҖкё°мҙҲкё° нҷ”л©ҙ л ҢлҚ”л§Ғ
failureChallenge(error, Response)мӢӨнҢЁ л©”мӢңм§ҖмҷҖ н•Ёк»ҳ к°ҷмқҖ нҸј мһ¬л…ём¶ңмһҳлӘ»лҗң мӮ¬лІҲ мһ…л Ҙ л“ұ мһ¬мӢңлҸ„ н—Ҳмҡ©
failure(AuthenticationFlowError)м№ҳлӘ… мӢӨнҢЁ вҶ’ лЎңк·ёмқё мў…лЈҢм—°мҶҚ мӢӨнҢЁ н•ңлҸ„ мҙҲкіј
attempted()"мӢңлҸ„н–Ҳм§Җл§Ң лӮҙ мұ…мһ„мқҖ м•„лӢҳ" вҶ’ к°ҷмқҖ л ҲлІЁ лӢӨмқҢ AlternativeлЎңмЎ°кұҙ лҜён•ҙлӢ№мңјлЎң нҢЁмҠӨ
forceChallenge(Response)Requiredм—¬лҸ„ мһ¬мӢңлҸ„ л¶Ҳк°ҖлҠҘн•ң лӢЁмқј Challengeм•ҪкҙҖ лҸҷмқҳмІҳлҹј л¬ҙмЎ°кұҙ ліҙм—¬мЈјлҠ” нҷ”л©ҙ

attempted()мҷҖ failure()мқҳ м°Ёмқҙк°Җ мҙҲмӢ¬мһҗк°Җ к°ҖмһҘ мһҗмЈј нӢҖлҰ¬лҠ” л¶Җл¶„мқҙлӢӨ. мӮ¬мҡ©мһҗк°Җ м»ӨмҠӨн…Җ мқёмҰқ мҲҳлӢЁмқ„ "м„ нғқн•ҳм§Җ м•ҠмқҖ" кІҪмҡ°лҠ” attempted()лӢӨ. мӢңлҸ„н–ҲлҠ”лҚ° "мһҳлӘ»лҗң" кІҪмҡ°лҠ” failureChallenge()лӮҳ failure()лӢӨ.

3. нҸј л ҢлҚ”л§Ғ вҖ” FreeMarker н…ңн”ҢлҰҝ

мӮ¬мҡ©мһҗм—җкІҢ мһ…л Ҙмқ„ л°ӣмңјл Өл©ҙ HTML нҸјмқ„ к·ёл Өм•ј н•ңлӢӨ. KeycloakмқҖ FreeMarker кё°л°ҳ LoginFormsProviderлҘј м ңкіөн•ңлӢӨ.

кё°ліё нҢЁн„ҙ

@Override
public void authenticate(AuthenticationFlowContext context) {
    Response challenge = context.form()
        .setAttribute("employeeNumber", "")
        .createForm("employee-form.ftl");
    context.challenge(challenge);
}

@Override
public void action(AuthenticationFlowContext context) {
    MultivaluedMap<String, String> form =
        context.getHttpRequest().getDecodedFormParameters();
    String employeeNumber = form.getFirst("employeeNumber");

    if (employeeNumber == null || employeeNumber.isBlank()) {
        Response challenge = context.form()
            .setError("missingEmployeeNumber")
            .createForm("employee-form.ftl");
        context.failureChallenge(AuthenticationFlowError.INVALID_CREDENTIALS, challenge);
        return;
    }

    // кІҖмҰқ лЎңм§ҒмқҖ лӢӨмқҢ м Ҳм—җм„ң
}

@Override
public List<ProviderConfigProperty> getConfigProperties() {
    ProviderConfigProperty apiUrl = new ProviderConfigProperty();
    apiUrl.setName("apiUrl");
    apiUrl.setLabel("Employee API URL");
    apiUrl.setType(ProviderConfigProperty.STRING_TYPE);
    apiUrl.setHelpText("мӮ¬лІҲ кІҖмҰқ API лІ мқҙмҠӨ URL");
    apiUrl.setDefaultValue("https://hr.internal/api");

    timeout.setLabel("Timeout (ms)");
    timeout.setType(ProviderConfigProperty.STRING_TYPE);
    timeout.setDefaultValue("3000");

}

    @Override

        return new Requirement[] { Requirement.REQUIRED, Requirement.DISABLED };
    }

    @Override

Authenticator

package com.example.keycloak.employee;

import jakarta.ws.rs.core.MultivaluedMap;
import jakarta.ws.rs.core.Response;
import org.keycloak.authentication.AuthenticationFlowContext;
import org.keycloak.authentication.AuthenticationFlowError;
import org.keycloak.authentication.Authenticator;
import org.keycloak.models.*;

public class EmployeeAuthenticator implements Authenticator {

    private final EmployeeApiClient apiClient;

    public EmployeeAuthenticator(EmployeeApiClient apiClient) {
        this.apiClient = apiClient;
    }

    @Override
    public void authenticate(AuthenticationFlowContext context) {
        Response challenge = context.form()
            .setAttribute("employeeNumber", "")
            .createForm("employee-form.ftl");
        context.challenge(challenge);
    }

    @Override
    public void action(AuthenticationFlowContext context) {
        MultivaluedMap<String, String> form =
            context.getHttpRequest().getDecodedFormParameters();
        String employeeNumber = form.getFirst("employeeNumber");

        if (employeeNumber == null || employeeNumber.isBlank()) {
            context.failureChallenge(
                AuthenticationFlowError.INVALID_CREDENTIALS,
                context.form()
                    .setError("missingEmployeeNumber")
                    .createForm("employee-form.ftl"));
            return;
        }

        AuthenticatorConfigModel config = context.getAuthenticatorConfig();
        String apiUrl = config.getConfig().get("apiUrl");

        EmployeeApiClient.Result result = apiClient.verify(apiUrl, employeeNumber);
        if (!result.valid()) {
            context.failureChallenge(
                AuthenticationFlowError.INVALID_CREDENTIALS,
                context.form()
                    .setError("invalidEmployeeNumber")
                    .createForm("employee-form.ftl"));
            return;
        }

        // л¶Җм„ң м •ліҙлҘј User AttributeлЎң м ҖмһҘ
        UserModel user = context.getUser();
        user.setSingleAttribute("employeeNumber", employeeNumber);
        user.setSingleAttribute("department", result.department());

        context.success();
    }

    @Override
    public boolean requiresUser() { return true; }

    @Override
    public boolean configuredFor(KeycloakSession session, RealmModel realm, UserModel user) {
        return true;
    }

    @Override
    public void setRequiredActions(KeycloakSession session, RealmModel realm, UserModel user) { }

    @Override
    public void close() { }
}
