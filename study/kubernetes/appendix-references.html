부록 C. 참고 자료

스터디를 마친 뒤 더 깊이 파고들 수 있는 공식 문서·표준·심화 자료를 카테고리별로 모았다. 1차 출처는 언제나 공식 문서이므로 그것부터 둔다.

쿠버네티스 공식 문서 — 주요 섹션

  • Kubernetes Documentation (홈) — 전체 문서의 진입점
  • Concepts — 아키텍처·오브젝트·동작 원리 개념 전반
  • Workloads — Pod와 워크로드 컨트롤러
  • Services, Load Balancing, and Networking — Service·Ingress·네트워크 정책
  • Storage — Volume·PV·PVC·StorageClass·CSI
  • Configuration — ConfigMap·Secret·리소스 관리
  • Security — 보안 개념 전반
  • Authentication / Authorization (RBAC) — 인증·인가·RBAC
  • Scheduling, Preemption and Eviction — 스케줄러·affinity·taint
  • Extending Kubernetes (CRD/Operator) — CRD·API 확장·Operator
  • Tasks — 목적별 실습 가이드 모음
  • Debug Application / Cluster — 트러블슈팅 절차
  • Reference (API / kubectl) — API 레퍼런스와 kubectl 명령어
  • kubectl Cheatsheet — 공식 kubectl 치트시트
  • Standardized Glossary — 공식 용어집
  • Production-Grade Container Orchestration — 프로젝트 메인

컨테이너 / Docker

  • Docker Documentation — Docker 전체 문서
  • Dockerfile reference — Dockerfile 명령어 레퍼런스
  • Building best practices — 이미지 빌드 모범 사례
  • Docker Compose — 로컬 멀티 컨테이너 구성
  • containerd — 대표적인 컨테이너 런타임
  • CRI-O — 쿠버네티스 전용 경량 런타임

표준 / 스펙

  • OCI (Open Container Initiative) — 컨테이너 표준화 단체
  • OCI Image Spec — 이미지 포맷 표준
  • OCI Runtime Spec — 런타임 표준
  • CNI (Container Network Interface) — 네트워크 플러그인 표준
  • CSI (Container Storage Interface) — 스토리지 플러그인 표준
  • Kubernetes API Reference — 모든 API 오브젝트 스펙

CNCF / 생태계

  • Cloud Native Computing Foundation — 클라우드 네이티브 생태계를 관장하는 재단
  • CNCF Landscape — 클라우드 네이티브 도구 지도
  • CNCF Projects — 졸업·인큐베이팅 프로젝트 목록
  • 12-Factor App — 클라우드 네이티브 애플리케이션 설계 원칙

패키징 / GitOps / 서비스 메시

  • Helm — 쿠버네티스 패키지 매니저
  • Kustomize — 매니페스트 오버레이 기반 커스터마이징
  • Argo CD — 선언적 GitOps 지속적 배포
  • Argo Workflows — 쿠버네티스 네이티브 워크플로 엔진
  • Istio — 대표적인 서비스 메시
  • Cilium — eBPF 기반 CNI·네트워크 보안
  • cert-manager — 인증서 자동 발급·갱신
  • Prometheus — 모니터링·알림 표준 도구

심화 — KEP / 블로그 / 학습 자료

  • Kubernetes Enhancement Proposals (KEP) — 기능 제안·설계 문서 저장소
  • Kubernetes Blog — 릴리스 노트·심화 글
  • Kubernetes Community — SIG·기여 가이드
  • The Kubernetes Book / Resources 모음 (CNCF) — CNCF 온라인 프로그램·웨비나
  • Kubernetes the Hard Way — 클러스터를 수작업으로 구성하며 내부를 이해하는 실습 자료
  • Kubernetes Failure Stories — 실제 장애 사례 모음. 트러블슈팅 학습에 유용
