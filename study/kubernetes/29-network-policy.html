NetworkPolicy

н•ҷмҠө лӘ©н‘ң

  • кё°ліё мғҒнғңк°Җ "лӘЁл‘җ н—Ҳмҡ©"мқё мқҙмң мҷҖ NetworkPolicyлЎң "кұ°л¶Җ"лЎң м „нҷҳн•ҳлҠ” мӣҗлҰ¬лҘј мқҙн•ҙн•ңлӢӨ.
  • ingressмҷҖ egress к·ңм№ҷмқҳ л°©н–Ҙкіј мқҳлҜёлҘј м •нҷ•нһҲ м•ҲлӢӨ.
  • podSelectorВ·namespaceSelectorВ·ipBlockмңјлЎң лҢҖмғҒмқ„ м–ҙл–»кІҢ м§Җм •н•ҳлҠ”м§Җ мқөнһҢлӢӨ.
  • NetworkPolicyк°Җ CNIм—җ мқҳмЎҙн•ңлӢӨлҠ” м җкіј к·ё мқҳлҜёлҘј нҢҢм•…н•ңлӢӨ.

1. NetworkPolicy к°ңл…җ вҖ” default allowм—җм„ң denyлЎң

мҝ лІ„л„ӨнӢ°мҠӨмқҳ кё°ліё мғҒнғңм—җм„ңлҠ” лӘЁл“  Podк°Җ лӘЁл“  PodмҷҖ мһҗмң лЎӯкІҢ нҶөмӢ н•  мҲҳ мһҲлӢӨ(default allow). мҰү кІ©лҰ¬к°Җ м—ҶлӢӨ. мқҙлҠ” нҺёлҰ¬н•ҳм§Җл§Ң, м№Ён•ҙлҗң Pod н•ҳлӮҳк°Җ нҒҙлҹ¬мҠӨн„° м „мІҙлҘј мһҗмң лЎӯкІҢ нғҗмғүн•  мҲҳ мһҲлӢӨлҠ” ліҙм•Ҳ мң„н—ҳмқ„ лң»н•ңлӢӨ.

NetworkPolicyлҠ” Pod мҲҳмӨҖмқҳ л°©нҷ”лІҪ к·ңм№ҷмқҙлӢӨ. н•өмӢ¬ лҸҷмһ‘ мӣҗлҰ¬лҠ” лӢӨмқҢкіј к°ҷлӢӨ.

  • м–ҙл–Ө Podм—җ NetworkPolicyк°Җ н•ҳлӮҳлқјлҸ„ м Ғмҡ©лҗҳл©ҙ, к·ё PodлҠ” н•ҙлӢ№ л°©н–Ҙ(ingress/egress)м—җ лҢҖн•ҙ кё°ліё кұ°л¶Җ(deny) мғҒнғңк°Җ лҗңлӢӨ.
  • к·ё л’ӨлЎңлҠ” м •мұ…м—җм„ң лӘ…мӢңм ҒмңјлЎң н—Ҳмҡ©н•ң нҠёлһҳн”Ҫл§Ң нҶөкіјн•ңлӢӨ.
  • м •мұ…мқҖ нҷ”мқҙнҠёлҰ¬мҠӨнҠё л°©мӢқмқҙлӢӨ. "кұ°л¶Җ" к·ңм№ҷмқҖ м—Ҷкі , н—Ҳмҡ© к·ңм№ҷмқҳ н•©м§‘н•©л§Ң мЎҙмһ¬н•ңлӢӨ.

м „мІҙ к°ңл…җмқҖ Network Policies л¬ём„ңм—җ м •лҰ¬лҸј мһҲлӢӨ.

2. default deny вҖ” кІ©лҰ¬мқҳ мӢңмһ‘

мўӢмқҖ ліҙм•Ҳ м¶ңл°ңм җмқҖ л„Өмһ„мҠӨнҺҳмқҙмҠӨ м „мІҙм—җ лӘЁл“  мқёк·ёл ҲмҠӨлҘј кё°ліё кұ°л¶Җн•ҳлҠ” м •мұ…мқ„ к№”кі , н•„мҡ”н•ң нҶөмӢ л§Ң м¶”к°ҖлЎң н—Ҳмҡ©н•ҳлҠ” кІғмқҙлӢӨ. лӢӨмқҢ м •мұ…мқҖ default л„Өмһ„мҠӨнҺҳмқҙмҠӨмқҳ лӘЁл“  Podм—җ лҢҖн•ҙ л“Өм–ҙмҳӨлҠ” нҠёлһҳн”Ҫмқ„ м „л¶Җ л§үлҠ”лӢӨ.

apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: default-deny-ingress
  namespace: default
spec:
  podSelector: {}        # л№Ҳ м…Җл үн„° = л„Өмһ„мҠӨнҺҳмқҙмҠӨмқҳ лӘЁл“  Pod
  policyTypes:
  - Ingress
  # ingress к·ңм№ҷмқҙ м—ҶмңјлҜҖлЎң лӘЁл“  мқёк·ёл ҲмҠӨ кұ°л¶Җ

podSelector: {}лҠ” л„Өмһ„мҠӨнҺҳмқҙмҠӨмқҳ лӘЁл“  PodлҘј лҢҖмғҒмңјлЎң н•ҳкі , IngressлҘј policyTypesм—җ л„Јм—Ҳм§Җл§Ң н—Ҳмҡ© к·ңм№ҷ(ingress:)мқ„ л№„мӣҢ л‘җм—ҲмңјлҜҖлЎң кІ°кіјм ҒмңјлЎң лӘЁл“  мқёк·ёл ҲмҠӨлҘј м°ЁлӢЁн•ңлӢӨ. egressк№Ңм§Җ л§үмңјл Өл©ҙ policyTypesм—җ EgressлҘј м¶”к°Җн•ңлӢӨ.

spec:
  podSelector: {}
  policyTypes:
  - Ingress
  - Egress
# м–‘л°©н–Ҙ лӘЁл‘җ кё°ліё кұ°л¶Җ

egress м°ЁлӢЁ мӢң DNSлҸ„ л§үнһҢлӢӨ

egressлҘј м „л¶Җ л§үмңјл©ҙ Podк°Җ CoreDNS(53/UDP, 53/TCP)м—җлҸ„ м ‘к·јн•ҳм§Җ лӘ»н•ҙ мқҙлҰ„ мЎ°нҡҢк°Җ к№Ём§„лӢӨ. egress м •мұ…мқ„ лҸ„мһ…н•  л•ҢлҠ” кұ°мқҳ н•ӯмғҒ kube-systemмқҳ DNSлЎң к°ҖлҠ” нҠёлһҳн”Ҫмқ„ лӘ…мӢңм ҒмңјлЎң н—Ҳмҡ©н•ҙм•ј н•ңлӢӨ.

3. ingressмҷҖ egress к·ңм№ҷ

NetworkPolicy к·ңм№ҷмқҖ л°©н–Ҙмқҙ л¶„лӘ…н•ҳлӢӨ.

  • ingress: лҢҖмғҒ PodлЎң л“Өм–ҙмҳӨлҠ” нҠёлһҳн”Ҫ. fromмңјлЎң м¶ңл°ңм§ҖлҘј м§Җм •н•ңлӢӨ.
  • egress: лҢҖмғҒ Podм—җм„ң лӮҳк°ҖлҠ” нҠёлһҳн”Ҫ. toлЎң лӘ©м Ғм§ҖлҘј м§Җм •н•ңлӢӨ.

    src["м¶ңл°ңм§Җ Pod"] -- "ingress: fromмңјлЎң н—Ҳмҡ©" --> target["лҢҖмғҒ Pod<br>(podSelector)"]
    target -- "egress: toлЎң н—Ҳмҡ©" --> dst["лӘ©м Ғм§Җ"]

лӢӨмқҢмқҖ app=webмқё PodлЎң, app=frontendмқё Podм—җм„ң мҳӨлҠ” 8080 нҠёлһҳн”Ҫл§Ң н—Ҳмҡ©н•ҳлҠ” м •мұ…мқҙлӢӨ.

apiVersion: networking.k8s.io/v1
kind: NetworkPolicy
metadata:
  name: web-allow-frontend
  namespace: default
spec:
  podSelector:
    matchLabels:
      app: web              # мқҙ м •мұ…мқҙ м Ғмҡ©лҗҳлҠ” лҢҖмғҒ Pod
  policyTypes:
  - Ingress
  ingress:
  - from:
    - podSelector:
        matchLabels:
          app: frontend     # frontend Podм—җм„ң мҳӨлҠ” кІғл§Ң
    ports:
    - protocol: TCP
      port: 8080

мқҙ м •мұ…мқҙ web Podм—җ м Ғмҡ©лҗҳлҠ” мҲңк°„, frontendк°Җ м•„лӢҢ лӢӨлҘё Podм—җм„ң мҳӨлҠ” нҠёлһҳн”ҪмқҖ лӘЁл‘җ кұ°л¶ҖлҗңлӢӨ.

4. м…Җл үн„° вҖ” podSelector, namespaceSelector, ipBlock

from/toм—җм„ң м¶ңл°ңм§ҖВ·лӘ©м Ғм§ҖлҘј м§Җм •н•ҳлҠ” л°©лІ•мқҖ м„ё к°Җм§ҖлӢӨ.

podSelector. к°ҷмқҖ л„Өмһ„мҠӨнҺҳмқҙмҠӨ м•Ҳм—җм„ң л Ҳмқҙлё”лЎң PodлҘј кі лҘёлӢӨ.

  ingress:
  - from:
    - podSelector:
        matchLabels:
          role: api

namespaceSelector. л Ҳмқҙлё”мқҙ л§һлҠ” л„Өмһ„мҠӨнҺҳмқҙмҠӨ м „мІҙлҘј кі лҘёлӢӨ. лӢӨлҘё л„Өмһ„мҠӨнҺҳмқҙмҠӨм—җм„ң мҳӨлҠ” нҠёлһҳн”Ҫмқ„ н—Ҳмҡ©н•  л•Ң м“ҙлӢӨ.

  ingress:
  - from:
    - namespaceSelector:
        matchLabels:
          team: payment

ipBlock. CIDRлЎң IP лҢҖм—ӯмқ„ кі лҘёлӢӨ. нҒҙлҹ¬мҠӨн„° мҷёл¶Җ мЈјмҶҢлҘј н—Ҳмҡ©В·м°ЁлӢЁн•  л•Ң м“ҙлӢӨ. exceptлЎң мқјл¶ҖлҘј л№јлӮј мҲҳ мһҲлӢӨ.

  egress:
  - to:
    - ipBlock:
        cidr: 10.0.0.0/8
        except:
        - 10.0.5.0/24      # мқҙ лҢҖм—ӯл§Ң м ңмҷё

podSelectorмҷҖ namespaceSelectorмқҳ AND vs OR

н•ң from н•ӯлӘ© м•Ҳм—җм„ң podSelectorмҷҖ namespaceSelectorлҘј к°ҷмқҖ лҰ¬мҠӨнҠё мҡ”мҶҢм—җ н•Ёк»ҳ м“°л©ҙ AND(л‘ҳ лӢӨ л§ҢмЎұ)мқҙкі , лі„лҸ„ лҰ¬мҠӨнҠё мҡ”мҶҢлЎң лӮҳлҲ„л©ҙ ORмқҙлӢӨ. л“Өм—¬м“°кё° н•ң м№ё м°ЁмқҙлЎң мқҳлҜёк°Җ м •л°ҳлҢҖк°Җ лҗҳлҜҖлЎң л§Өмҡ° мЈјмқҳн•ңлӢӨ.

  # AND: payment л„Өмһ„мҠӨнҺҳмқҙмҠӨмқҳ role=api Pod
  ingress:
  - from:
    - namespaceSelector:
        matchLabels:
          team: payment
      podSelector:               # к°ҷмқҖ мҡ”мҶҢ(- м—ҶмқҢ) вҶ’ AND
        matchLabels:
          role: api

  # OR: payment л„Өмһ„мҠӨнҺҳмқҙмҠӨмқҳ лӘЁл“  Pod лҳҗлҠ” role=api Pod
  ingress:
  - from:
    - namespaceSelector:
        matchLabels:
          team: payment
    - podSelector:               # лі„лҸ„ мҡ”мҶҢ(- мһҲмқҢ) вҶ’ OR
        matchLabels:
          role: api

5. CNI м§Җмӣҗ мҡ”кұҙкіј мў…н•© мҳҲм ң

м—¬кё°м„ң л°ҳл“ңмӢң м•Ңм•„м•ј н•  м җмқҖ, NetworkPolicyлҠ” CNI н”Ңлҹ¬к·ёмқёмқҙ мӢӨм ңлЎң мӢңн–ү(enforce)н•ҙм•јл§Ң лҸҷмһ‘н•ңлӢӨлҠ” кІғмқҙлӢӨ. мҝ лІ„л„ӨнӢ°мҠӨ APIлҠ” NetworkPolicy мҳӨлёҢм қнҠёлҘј л°ӣм•„ м ҖмһҘн•ҳм§Җл§Ң, к·ёкІғмқ„ нҢЁнӮ· мҲҳмӨҖм—җм„ң л§үлҠ” мқјмқҖ CNIк°Җ н•ңлӢӨ. NetworkPolicyлҘј м§Җмӣҗн•ҳм§Җ м•ҠлҠ” CNI(мҳҲ: кё°ліё Flannel)м—җм„ңлҠ” м •мұ…мқ„ л§Ңл“Өм–ҙлҸ„ м•„л¬ҙ нҡЁкіј м—Ҷмқҙ лӘЁл“  нҠёлһҳн”Ҫмқҙ нҶөкіјн•ңлӢӨ.
