용어집

이 스터디에서 다룬 용어를 분야별로 묶어 정리한다. 각 항목에는 한 줄 설명과 관련 챕터 링크가 포함된다.

사용법

영문 약어는 약어 기준으로 정렬했다. 한글 용어는 괄호 안에 병기한다. 관련 챕터가 여러 개면 주요 챕터 하나만 표기하고, 전체 목차는 참고 자료에서 확인한다.

아래 다이어그램은 이 스터디의 핵심 개념 계층을 나타낸다. 용어 간 관계를 파악하는 데 활용한다.

임베디드 기반

용어한글 명칭한 줄 설명관련 챕터
HAL하드웨어 추상화 계층Hardware Abstraction Layer. 보드 종류에 무관하게 동일한 API로 GPIO·SPI·I2C·UART 등 하드웨어를 제어할 수 있게 감싸는 레이어. AP_HAL 네임스페이스로 구현된다.CH4. HAL
RTOS실시간 운영체제Real-Time Operating System. 태스크 스케줄링·인터럽트 지연을 확정적으로(deterministic) 보장하는 임베디드 OS. ArduPilot은 ChibiOS를 기본 RTOS로 사용한다.CH5. 보드와 RTOS
ChibiOS—Pixhawk 계열 STM32 보드에서 ArduPilot이 채택한 오픈소스 RTOS. 스레드·세마포어·타이머 등 POSIX 유사 API를 제공한다.CH5. 보드와 RTOS
MCU마이크로컨트롤러Microcontroller Unit. CPU·RAM·Flash·주변장치를 하나의 칩에 통합한 소형 프로세서. Pixhawk 4는 STM32H743 기반이다.CH5. 보드와 RTOS
펌웨어firmwareMCU 내 Flash에 올라가는 실행 이미지. ArduCopter.bin 등이 이에 해당한다.CH3. ArduPilot 개요
크로스컴파일cross-compilation타깃 아키텍처(ARM Cortex-M)와 다른 호스트(x86 PC)에서 펌웨어를 빌드하는 것. arm-none-eabi-gcc 툴체인을 사용한다.CH3. ArduPilot 개요
실시간성real-time property정해진 데드라인 안에 태스크가 완료되는 성질. ArduPilot 메인 루프는 400 Hz(2.5 ms) 주기 안에 FAST_TASK를 반드시 처리해야 한다.CH8. 실시간 스케줄링
hwdefhardware definition보드 핀맵·버스 연결·드라이버 활성화를 선언하는 텍스트 파일(hwdef.dat). ChibiOS 빌드 시 HAL 소스가 이 파일을 파싱해 자동 생성된다.CH5. 보드와 RTOS

스케줄러

용어한글 명칭한 줄 설명관련 챕터
스케줄러schedulerAP_Scheduler 클래스. 태스크 테이블을 순회하며 각 태스크를 지정된 주파수와 우선순위로 실행한다.CH7. 스케줄러
FAST_TASK—매 루프마다 반드시 실행되어야 하는 태스크에 붙이는 매크로 플래그. IMU 읽기·자세제어 등이 해당한다. SCHED_TASK_CLASS에 FAST_TASK 플래그를 지정해 등록한다.CH7. 스케줄러
협조적 스케줄링cooperative scheduling각 태스크가 스스로 제어를 반환하는 방식의 스케줄링. 선점(preemptive)과 달리 태스크가 오래 실행되면 다른 태스크가 밀린다. ArduPilot 메인 루프 내에서 채택된다.CH8. 실시간 스케줄링
루프 주파수loop frequency스케줄러가 메인 루프를 초당 몇 번 반복하는지. ArduCopter 기본값은 400 Hz. 로터 수·보드 성능에 따라 200~400 Hz 범위로 조정 가능하다.CH7. 스케줄러

센서

용어한글 명칭한 줄 설명관련 챕터
IMU관성 측정 장치Inertial Measurement Unit. 자이로스코프와 가속도계를 묶은 장치. 각속도와 선가속도를 출력한다.CH10. IMU
자이로스코프gyroscope각속도(rad/s)를 측정하는 센서. MEMS 진동형이 주로 쓰인다. 바이어스 드리프트가 있어 단독으로는 자세를 오래 추정하지 못한다.CH10. IMU
가속도계accelerometer선가속도(m/s²)를 측정하는 센서. 정적 자세에서는 중력 방향을 알 수 있지만 진동·외란에 취약하다.CH10. IMU
FIFO—First In First Out. IMU 내부 버퍼. DMA 전송으로 여러 샘플을 한 번에 읽어 CPU 인터럽트 부하를 낮춘다.CH10. IMU
노치필터notch filter특정 주파수 대역의 신호만 감쇠하는 필터. 프로펠러 회전에 의한 진동 주파수를 억제해 자이로 노이즈를 낮춘다.CH11. 진동과 필터링
기압계barometer대기압을 측정해 고도를 추정하는 센서. 실외에서 ±1 m 수준 정확도를 가지나 바람에 영향받기 쉽다. AP_Baro 라이브러리가 담당한다.CH13. 기압계·나침반·거리센서
나침반compass / magnetometer지구 자기장을 측정해 기수 방향(yaw)을 추정하는 센서. 모터·전선의 자기 간섭이 크기 때문에 캘리브레이션이 필수다. AP_Compass 담당.CH13. 기압계·나침반·거리센서
GPS—Global Positioning System. GNSS 수신기로 위도·경도·고도·속도를 측정한다. 대기 오차로 수평 오차 2~5 m가 일반적이다. AP_GPS 담당.CH12. GPS
fixGPS fixGPS 수신기가 위치를 계산하기에 충분한 위성 신호를 잠금(lock)한 상태. 3D fix 이상이어야 비행이 가능하다.CH12. GPS
RTK실시간 이동 측량Real-Time Kinematic. 기준국(base station) 보정 신호를 활용해 센티미터급 위치 정확도를 달성하는 GPS 기법.CH12. GPS
RangeFinder거리 센서초음파·레이저 등으로 지면까지 거리를 직접 측정하는 센서. 저고도 착륙·충돌 회피에 사용된다. AP_RangeFinder 담당.CH13. 기압계·나침반·거리센서

상태 추정

용어한글 명칭한 줄 설명관련 챕터
센서 퓨전sensor fusion여러 센서의 측정값을 통계적으로 결합해 단일 센서보다 정확한 상태 추정값을 만드는 기법.CH14. 센서 퓨전 개론
상태추정state estimation드론의 자세·위치·속도 등 직접 측정하기 어려운 상태(state)를 관측값으로부터 계산하는 과정.CH14. 센서 퓨전 개론
NED북-동-하 좌표계North-East-Down. 항공에서 표준으로 사용하는 지구 고정 좌표계. x가 북, y가 동, z가 아래를 향한다.CH14. 센서 퓨전 개론
body frame기체 좌표계드론 본체에 고정된 좌표계. 전진 방향이 x, 우측이 y, 아래가 z다. IMU 측정값은 body frame 기준이다.CH14. 센서 퓨전 개론
쿼터니언quaternion4원수. 3D 회전을 나타내는 4개 원소(w, x, y, z)로 이루어진 수. 짐벌락이 없어 항공·로봇 자세 표현에 표준으로 사용된다.CH14. 센서 퓨전 개론
짐벌락gimbal lock오일러 각(roll/pitch/yaw) 표현에서 두 회전축이 정렬되어 자유도를 잃는 현상. 쿼터니언을 사용하면 회피할 수 있다.CH14. 센서 퓨전 개론
칼만필터Kalman filter상태 예측과 측정 갱신을 반복해 노이즈 환경에서 최적 상태를 추정하는 재귀 필터. EKF·UKF의 기반이다.CH15. 칼만필터
공분산covariance두 상태 변수 사이의 오차 상관도. 칼만필터에서 상태 불확실성을 P 행렬로 표현한다.CH15. 칼만필터
이노베이션innovation칼만필터에서 예측값과 실제 측정값의 차이(y = z − Hx̂). 이 값이 크면 예측 모델 혹은 센서에 문제가 있다는 신호다.CH15. 칼만필터
AHRS자세·기수 기준 시스템Attitude and Heading Reference System. 자이로+가속도계+나침반을 결합해 roll·pitch·yaw를 제공하는 시스템. AP_AHRS 라이브러리가 인터페이스를 제공한다.CH16. AHRS와 DCM
DCM방향 코사인 행렬Direction Cosine Matrix. 3×3 회전행렬로 기체 좌표계를 지구 좌표계로 변환한다. EKF 이전 ArduPilot의 자세 추정 주요 방식이었다.CH16. AHRS와 DCM
EKF3확장 칼만필터 3Extended Kalman Filter(version 3). ArduPilot 현행 기본 상태추정기. GPS·IMU·나침반·기압계 데이터를 퓨전해 위치·자세·속도를 추정한다. AP_NavEKF3 구현.CH17. EKF3 구조
laneEKF 레인EKF3가 여러 IMU에 대해 독립 인스턴스를 실행하는 구조. 한 레인이 이상 감지되면 다른 레인으로 자동 전환(failover)한다.CH17. EKF3 구조
GSF방위각 추정 필터GPS-based Yaw Source Filter. GPS 속도 벡터를 이용해 나침반 없이도 초기 yaw를 추정하는 필터. EKF3 내부에서 lane 초기화에 사용된다.CH18. EKF3 동작

제어

용어한글 명칭한 줄 설명관련 챕터
PID비례-적분-미분 제어기Proportional-Integral-Derivative. 현재 오차(P), 누적 오차(I), 오차 변화율(D)을 선형 결합해 제어 출력을 만드는 고전 제어기.CH19. PID 제어
P항비례항현재 오차에 비례한 출력. error × kP.CH19. PID 제어
I항적분항오차 시간 적분에 비례한 출력. 정상 상태 오차를 제거한다. ∫error dt × kI.CH19. PID 제어
D항미분항오차 변화율에 비례한 출력. 오버슈트를 억제하는 제동 역할을 한다. d(error)/dt × kD.CH19. PID 제어
와인드업integrator windupI항 적분기가 포화 상태에서도 계속 누적되어 매우 큰 제어 출력을 내는 현상. IMAX 클리핑과 방향 검사로 방지한다.CH19. PID 제어
캐스케이드 제어cascade control외부 루프(위치/속도)의 출력이 내부 루프(자세/각속도)의 목표값으로 입력되는 계층형 제어 구조. 각 루프는 서로 다른 주파수로 동작한다.CH20. 캐스케이드 제어
자세제어attitude controlroll·pitch·yaw 각도를 목표값으로 유지하는 제어 루프. AC_AttitudeControl이 담당한다.CH21. 자세제어
위치제어position controlGPS·EKF3 위치 추정값을 이용해 드론이 특정 위치에 머물거나 이동하도록 하는 제어 루프. AC_PosControl 담당.CH22. 위치·항법
웨이포인트waypoint미션에서 드론이 순서대로 방문해야 하는 지점. 위도·경도·고도로 정의한다.CH26. 자동 비행과 미션
S-curveS-커브 궤적웨이포인트 간 이동 시 가속도를 부드럽게 변화시켜 저크(jerk)를 제한하는 속도 프로파일. AC_WPNav가 구현한다.CH22. 위치·항법
모터 믹싱motor mixing자세제어 출력(roll/pitch/yaw/throttle)을 각 모터의 PWM 출력으로 변환하는 과정. AP_Motors 라이브러리가 담당한다.CH23. 모터 믹싱
ESC전자 속도 제어기Electronic Speed Controller. FC의 PWM·DSHOT 신호를 받아 BLDC 모터 회전수를 조절하는 장치.CH23. 모터 믹싱
