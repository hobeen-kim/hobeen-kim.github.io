1. л¶ҲліҖ нҒҙлһҳмҠӨлһҖ?

л¶ҲліҖ(Immutable) нҒҙлһҳмҠӨлҠ” н•ң лІҲ мғқм„ұлҗҳл©ҙ к°қмІҙ лӮҙл¶Җмқҳ мғҒнғң(к°’)лҘј ліҖкІҪн•  мҲҳ м—ҶлҠ” нҒҙлһҳмҠӨлҘј л§җн•ңлӢӨ. мқёмҠӨн„ҙмҠӨлҠ” мғқм„ұ мӢңм җм—җ лӘЁл“  мғҒнғңк°Җ кІ°м •лҗҳкі , к·ё мқҙнӣ„м—җлҠ” к°’мқҙ м ҲлҢҖ л°”лҖҢм§Җ м•ҠлҠ”лӢӨ. мғҒнғң ліҖкІҪмқҙ н•„мҡ”н•ҳлӢӨл©ҙ кё°мЎҙ мқёмҠӨн„ҙмҠӨлҘј мҲҳм •н•ҳлҠ” лҢҖмӢ  мғҲ мқёмҠӨн„ҙмҠӨлҘј мғҲлЎң л§Ңл“Өм–ҙ л°ҳнҷҳн•ҳлҠ” л°©мӢқмңјлЎң мІҳлҰ¬н•ңлӢӨ.

мһҗл°”м—җм„ң к°ҖмһҘ лҢҖн‘ңм Ғмқё л¶ҲліҖ нҒҙлһҳмҠӨлҠ” String мқҙлӢӨ. String мқҖ л¬ёмһҗм—ҙмқ„ л°”кҫјлӢӨлҠ” к°ңл…җмқҙлқјкі  лҠҗк»ҙм§ҖлҠ” м—¬лҹ¬ м—°мӮ°мқҙ мӢӨм ңлЎңлҠ” мғҲ л¬ёмһҗм—ҙ к°қмІҙлҘј л§Ңл“Өм–ҙ л°ҳнҷҳн•ңлӢӨ.

л¶ҲліҖ к°қмІҙлҠ” лӢӨмқҢкіј к°ҷмқҖ мһҘм җмқҙ мһҲлӢӨ.

  • Thread-safe : мғҒнғңк°Җ ліҖн•ҳм§Җ м•ҠмңјлӢҲ лҸҷкё°нҷ”(synchronized) м—ҶмқҙлҸ„ кіөмң н•ҳкё°к°Җ мүҪлӢӨ.
  • мҳҲмёЎ к°ҖлҠҘм„ұ : к°қмІҙк°Җ м–ҙл””м„ң м–ҙл–»кІҢ ліҖн–ҲлҠ”м§Җ м¶”м Ғн•  н•„мҡ”к°Җ м—Ҷм–ҙм„ң л””лІ„к№…мқҙ нҺён•ҳлӢӨ.
  • мәҗмӢұ/кіөмң м—җ мң лҰ¬ : к°ҷмқҖ к°’мқҙл©ҙ мһ¬мӮ¬мҡ©н•ҳкё° мүҪкі , н•ҙмӢң кё°л°ҳ м»¬л үм…ҳ(HashMap, HashSet)м—җм„ңлҸ„ м•Ҳм „н•ҳкІҢ м“ё мҲҳ мһҲлӢӨ.
  • л°©м–ҙм Ғ н”„лЎңк·ёлһҳл°Қм—җ лҸ„мӣҖ: мҷёл¶Җм—җм„ң лӘ°лһҳ к°’мқ„ л°”кҫёлҠ” л¶Җмһ‘мҡ©(side effect)мқ„ м°ЁлӢЁн•ңлӢӨ.

л°ҳлҢҖлЎң лӢЁм җлҸ„ мһҲлӢӨ.

  • мғҒнғң ліҖкІҪмқҙ мһҰмқҖ кІҪмҡ° к°қмІҙ мғқм„ұ л№„мҡ©кіј GC л¶ҖлӢҙмқҙ мҰқк°Җн•  мҲҳ мһҲлӢӨ.
  • м„Өкі„к°Җ мЎ°кёҲ лҚ” лІҲкұ°лЎӯкі (л°©м–ҙм Ғ ліөмӮ¬ л“ұ), м»¬л үм…ҳмқ„ лӢӨлЈ° л•Ң мӢ кІҪ м“ё нҸ¬мқёнҠёк°Җ лҠҳм–ҙлӮңлӢӨ.

2. л¶ҲліҖ нҒҙлһҳмҠӨ кө¬нҳ„

л¶ҲліҖ нҒҙлһҳмҠӨлҘј кө¬нҳ„н•ҳлҠ” кё°ліё к·ңм№ҷмқҖ лӢӨмқҢкіј к°ҷлӢӨ.

  • нҒҙлһҳмҠӨлҘј finalлЎң м„ м–ён•ҙ мғҒмҶҚмқ„ л§үлҠ”лӢӨ.
    • мғҒмҶҚмқҙ к°ҖлҠҘн•ҳл©ҙ н•ҳмң„ нҒҙлһҳмҠӨк°Җ лӮҙл¶Җ мғҒнғңлҘј ліҖкІҪ к°ҖлҠҘн•ң нҳ•нғңлЎң нҷ•мһҘн•ҙлІ„лҰҙ мҲҳ мһҲлӢӨ.
  • лӘЁл“  н•„л“ңлҘј private finalлЎң м„ м–ён•ңлӢӨ.
  • setterлҘј м ңкіөн•ҳм§Җ м•ҠлҠ”лӢӨ.
  • н•„л“ңк°Җ List, Map, л°°м—ҙмІҳлҹј к°ҖліҖ(mutable) к°қмІҙлқјл©ҙ,
    • мғқм„ұмһҗм—җм„ң defensive copy(л°©м–ҙм Ғ ліөмӮ¬)
    • getterм—җм„ңлҸ„ defensive copy лҘј м Ғмҡ©н•ҙ мҷёл¶Җм—җм„ң лӮҙл¶Җ мғҒнғңлҘј л°”кҫём§Җ лӘ»н•ҳкІҢ н•ңлӢӨ.
  • лӮҙл¶Җм—җм„ң мӮ¬мҡ©н•ҳлҠ” к°ҖліҖ н•„л“ңлҠ” к°ҖлҠҘн•ҳл©ҙ л¶ҲліҖ м»¬л үм…ҳмңјлЎң к°җмӢём„ң ліҙкҙҖн•ңлӢӨ.
  • equals/hashCode/toString мқ„ кө¬нҳ„н•ҙ к°’ к°қмІҙлЎңмқҳ мӮ¬мҡ©м„ұмқ„ лҶ’мқёлӢӨ.
public final class Item {
private final String name;
private final List<String> categories;

    public Item(String name, List<String> categories) {
        this.name = Objects.requireNonNull(name, "name must not be null");
        Objects.requireNonNull(categories, "categories must not be null");

        // defensive copy + (м„ нғқ) unmodifiableлЎң к°җмӢё лӮҙл¶Җ ліҙкҙҖ
        this.categories = Collections.unmodifiableList(new ArrayList<>(categories));
    }

    public String getName() {
        return name;
    }

    public List<String> getCategories() {
        // лӮҙл¶Җм—җм„ң unmodifiableлЎң ліҙкҙҖн•ҳкі  мһҲлӢӨл©ҙ к·ёлҢҖлЎң л°ҳнҷҳн•ҙлҸ„ м•Ҳм „
        return categories;
    }

    // "ліҖкІҪ"мқҙ н•„мҡ”н•ҳл©ҙ мғҲ к°қмІҙлҘј л§Ңл“Өм–ҙ л°ҳнҷҳ
    public Item withName(String newName) {
        return new Item(newName, this.categories);
    }

    public Item withAddedCategory(String category) {
        List<String> next = new ArrayList<>(this.categories);
        next.add(category);
        return new Item(this.name, next);
    }

    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (!(o instanceof Item)) return false;
        Item item = (Item) o;
        return name.equals(item.name) && categories.equals(item.categories);
    }

    @Override
    public int hashCode() {
        return Objects.hash(name, categories);
    }

    @Override
    public String toString() {
        return "Item{name='" + name + "', categories=" + categories + '}';
    }
}

2.1 getter м—җм„ңлҸ„ л°©м–ҙм Ғ ліөмӮ¬к°Җ н•„мҡ”н•ң мқҙмң 

л¶ҲліҖ к°қмІҙм—җм„ң getter лҘј л°©м–ҙм Ғ ліөмӮ¬н•ҳм§Җ м•Ҡмңјл©ҙ н•„л“ңмқҳ к°қмІҙ лӮҙл¶Җк°Җ ліҖкІҪлҗ  мҲҳлҸ„ мһҲлӢӨ. мҳҲлҘј л“Өм–ҙ лӮҙл¶Җ н•„л“ңлҘј м•„лһҳмІҳлҹј к·ёлғҘ л°ҳнҷҳн•ҳл©ҙ,

public List<String> getCategories() {
return categories; // мң„н—ҳ!
}

мҷёл¶Җм—җм„ң мқҙл ҮкІҢ л°”кҝҖ мҲҳ мһҲлӢӨ.

Item item = new Item("keyboard", List.of("input", "pc"));
item.getCategories().add("hacked"); // лӮҙл¶Җ лҰ¬мҠӨнҠёк°Җ mutableмқҙл©ҙ мғҒнғңк°Җ к№Ём§җ

к·ёлһҳм„ң л‘җ к°Җм§Җ м „лһө мӨ‘ н•ҳлӮҳлҘј м„ нғқн•ңлӢӨ.

м „лһө A - getterм—җм„ң л§ӨлІҲ ліөмӮ¬н•ҙм„ң л°ҳнҷҳ

return new ArrayList<>(categories);

лӢЁ, нҳём¶ңмқҙ мһҰмңјл©ҙ ліөмӮ¬ л№„мҡ©мқҙ мғқкёҙлӢӨ.

м „лһө B - мғқм„ұмһҗм—җм„ң л¶ҲліҖ м»¬л үм…ҳмңјлЎң л§Ңл“Өм–ҙ ліҙкҙҖн•ҳкі  к·ёлҢҖлЎң л°ҳнҷҳ

Collections.unmodifiableList(new ArrayList<>(categories))

нӣЁм”¬ нқ”н•ҳкІҢ м“°лҠ” л°©мӢқмқҙлӢӨ. (л°ҳнҷҳ л№„мҡ©мқҙ кұ°мқҳ м—ҶмқҢ). мң„ мҳҲмӢңлҠ” B лҘј м Ғмҡ©н–ҲлӢӨ.

мЈјмқҳ

Collections.unmodifiableList(...)лҠ” м§„м§ң л¶ҲліҖ лҰ¬мҠӨнҠёлҘј л§Ңл“ңлҠ” кІҢ м•„лӢҲлқј мҲҳм • л©”м„ңл“ң нҳём¶ңмқ„ л§үлҠ” л·°(view)лӢӨ. к·ёлһҳм„ң мӣҗліё лҰ¬мҠӨнҠёлҘј к·ёлҢҖлЎң к°җмӢёл©ҙ м•Ҳ лҗҳкі , л°ҳл“ңмӢң new ArrayList<>(...) к°ҷмқҖ ліөмӮ¬ліёмқ„ к°җмӢј л’Ө ліҙкҙҖн•ҙм•ј н•ңлӢӨ.

3. л¶ҲліҖ нҒҙлһҳмҠӨ кө¬нҳ„ мӢң мһҗмЈј лҶ“м№ҳлҠ” л¶Җл¶„

л¶ҲліҖ нҒҙлһҳмҠӨ кө¬нҳ„ мӢң мһҗмЈј лҶ“м№ҳлҠ” нҢЁн„ҙл“ӨмқҙлӢӨ.

3.1 л°°м—ҙ н•„л“ң (byte[], int[])лҠ” нҠ№нһҲ мң„н—ҳ

л°°м—ҙмқҖ лҢҖн‘ңм Ғмқё к°ҖліҖ к°қмІҙлқјм„ң л°©м–ҙм Ғ ліөмӮ¬к°Җ н•„мҲҳлӢӨ.


public final class Token {
private final byte[] value;

    public Token(byte[] value) {
        this.value = value.clone(); // defensive copy
    }
    
    public byte[] getValue() {
        return value.clone(); // defensive copy
    }
}

3.2 к°ҖліҖ к°қмІҙлҘј н•„л“ңлЎң л“Өкі  мһҲмңјл©ҙ к№ҠмқҖ л¶ҲліҖмқҙ м•„лӢҲлӢӨ

мҳҲлҘј л“Өм–ҙ LocalDateTime к°ҷмқҖ кұҙ л¶ҲліҖмқҙлқј кҙңм°®м§Җл§Ң, Date(java.util.Date)мІҳлҹј к°ҖліҖмқё к°қмІҙлҠ” к·ёлҢҖлЎң л“Өкі  мһҲмңјл©ҙ мң„н—ҳн•ҳлӢӨ. мқҙлҹ° кІҪмҡ°лҸ„ л°©м–ҙм Ғ ліөмӮ¬к°Җ н•„мҡ”н•ҳлӢӨ.

public final class Event {
private final java.util.Date when;

    public Event(Date when) {
        this.when = new java.util.Date(when.getTime());
    }
    
    public Date getWhen() {
        return new java.util.Date(when.getTime());
    }
}

3.3. м»¬л үм…ҳ м•Ҳмқҳ мҡ”мҶҢк°Җ к°ҖліҖмқј л•Ң

List Person м—җм„ң Personмқҙ mutableмқҙл©ҙ, лҰ¬мҠӨнҠё мһҗмІҙлҠ” лӘ» л°”кҝ”лҸ„ мҡ”мҶҢлҠ” л°”лҖ” мҲҳ мһҲлӢӨ.

  • м–•мқҖ л¶ҲліҖ(shallow immutability): м»¬л үм…ҳ кө¬мЎ°л§Ң м•Ҳ л°”лҖң (м¶”к°Җ/мӮӯм ң л¶Ҳк°Җ)
  • к№ҠмқҖ л¶ҲліҖ(deep immutability): мҡ”мҶҢ к°қмІҙк№Ңм§Җ л¶ҲліҖ

к№ҠмқҖ л¶ҲліҖмқҙ н•„мҡ”н•ҳлӢӨл©ҙ, мҡ”мҶҢ нғҖмһ…лҸ„ л¶ҲліҖмңјлЎң м„Өкі„н•ҳкұ°лӮҳ, мҡ”мҶҢлҘј ліөмӮ¬н•ҙм„ң л“Өкі  мһҲм–ҙм•ј н•ңлӢӨ.

4. л§Ҳл¬ҙлҰ¬

л¶ҲліҖ нҒҙлһҳмҠӨлҠ” лӢЁмҲңнһҲ finalмқ„ л¶ҷмқҙлҠ” мҲҳмӨҖмқҙ м•„лӢҲлқј, мҷёл¶Җм—җм„ң лӮҙл¶Җ мғҒнғңм—җ м ‘к·јн•ҙ л°”кҝҖ мҲҳ мһҲлҠ” лӘЁл“  кІҪлЎңлҘј м°ЁлӢЁн•ҳлҠ” кІғмқҙ н•өмӢ¬мқҙлӢӨ. нҠ№нһҲ м»¬л үм…ҳ/л°°м—ҙ/к°ҖліҖ нғҖмһ… н•„л“ңм—җм„ңлҠ” л°©м–ҙм Ғ ліөмӮ¬лҘј мҠөкҙҖмІҳлҹј м Ғмҡ©н•ҙм•ј н•ңлӢӨ.

м •лҰ¬н•ҳл©ҙ,

  • private final н•„л“ң

  • ліҖкІҪмқҙ н•„мҡ”н•ҳл©ҙ withXxx()мІҳлҹј мғҲ к°қмІҙ л°ҳнҷҳ
