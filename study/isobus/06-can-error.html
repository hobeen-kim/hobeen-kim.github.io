  • TEC, REC м№ҙмҡҙн„°мқҳ мҰқк°җ к·ңм№ҷмқ„ м Ғмҡ©н•  мҲҳ мһҲлӢӨ

1. м—җлҹ¬ кІҖм¶ң л©”м»ӨлӢҲмҰҳ 5к°Җм§Җ

CANмқҖ н•ҳл“ңмӣЁм–ҙ л ҲлІЁм—җм„ң 5к°Җм§Җ лҸ…лҰҪм Ғмқё л°©лІ•мңјлЎң мҳӨлҘҳлҘј к°җм§Җн•ңлӢӨ. мқҙ мӨ‘ н•ҳлӮҳлқјлҸ„ мҳӨлҘҳлҘј к°җм§Җн•ҳл©ҙ мҰүмӢң м—җлҹ¬ н”„л Ҳмһ„мқ„ м „мҶЎн•ңлӢӨ.

1-1. Bit Error (л№„нҠё м—җлҹ¬)

мӣҗлҰ¬: л…ёл“ңк°Җ л№„нҠёлҘј м „мҶЎн•ҳлҠ” лҸҷмӢңм—җ лІ„мҠӨлҘј мқҪм–ҙм„ң, мһҗмӢ мқҙ ліҙлӮё к°’кіј лІ„мҠӨмқҳ к°’мқҙ лӢӨлҘҙл©ҙ мҳӨлҘҳлЎң нҢҗлӢЁн•ңлӢӨ.

м „мҶЎн•ң к°’: 1 (Recessive)
лІ„мҠӨ мқҪмқҖ к°’: 0 (Dominant) вҶҗ лӢӨлҘё л…ёл“ңк°Җ 0мқ„ ліҙлғ„
вҶ’ Bit Error л°ңмғқ

мҳҲмҷё: мӨ‘мһ¬(Arbitration) кө¬к°„кіј ACK мҠ¬лЎҜм—җм„ңлҠ” Bit ErrorлҘј кІҖмӮ¬н•ҳм§Җ м•ҠлҠ”лӢӨ. мӨ‘мһ¬ мӨ‘ нҢЁл°°н•ҳлҠ” кІғмқҖ м •мғҒ лҸҷмһ‘мқҙкё° л•Ңл¬ёмқҙлӢӨ.

1-2. Stuff Error (мҠӨн„°н”„ м—җлҹ¬)

мӣҗлҰ¬: л№„нҠё мҠӨн„°н•‘ к·ңм№ҷм—җ л”°лҘҙл©ҙ, к°ҷмқҖ к°’мқҳ л№„нҠё 5к°ң нӣ„м—җлҠ” л°ҳл“ңмӢң л°ҳлҢҖ л№„нҠёк°Җ мҳЁлӢӨ. мқҙ к·ңм№ҷмқҙ мң„л°ҳлҗҳл©ҙ Stuff Errorк°Җ л°ңмғқн•ңлӢӨ.

мҲҳмӢ лҗң л№„нҠём—ҙ: 1 1 1 1 1 1 ...
                          вҶ‘ 6лІҲм§ё лҸҷмқј л№„нҠё вҶ’ Stuff Error

1-3. CRC Error (CRC м—җлҹ¬)

мӣҗлҰ¬: мҶЎмӢ  л…ёл“ңк°Җ кі„мӮ°н•ң CRC к°’мқ„ н”„л Ҳмһ„м—җ нҸ¬н•Ён•ҙ м „мҶЎн•ңлӢӨ. мҲҳмӢ  л…ёл“ңлҠ” л°ӣмқҖ лҚ°мқҙн„°лЎң м§Ғм ‘ CRCлҘј кі„мӮ°н•ҙ н”„л Ҳмһ„мқҳ CRC н•„л“ңмҷҖ л№„көҗн•ңлӢӨ. к°’мқҙ лӢӨлҘҙл©ҙ CRC ErrorлӢӨ.

мҶЎмӢ  л…ёл“ң кі„мӮ° CRC: 0x1A3F
мҲҳмӢ  л…ёл“ң кі„мӮ° CRC: 0x1A3E  вҶҗ м „мҶЎ мӨ‘ л№„нҠё ліҖнҳ•
вҶ’ CRC Error л°ңмғқ

1-4. Form Error (нҸј м—җлҹ¬)

мӣҗлҰ¬: CAN н”„л Ҳмһ„мқҳ нҠ№м • н•„л“ң(CRC Delimiter, ACK Delimiter, EOF)лҠ” л°ҳл“ңмӢң Recessive(1)м—¬м•ј н•ңлӢӨ. мқҙ мң„м№ҳм—җм„ң Dominant(0)лҘј мҲҳмӢ н•ҳл©ҙ Form ErrorлӢӨ.

EOF н•„л“ң (л°ҳл“ңмӢң 7к°ң Recessive):
мҲҳмӢ : 1 1 1 0 1 1 1
              вҶ‘ Dominant вҶ’ Form Error

1-5. ACK Error (ACK м—җлҹ¬)

мӣҗлҰ¬: мҶЎмӢ  л…ёл“ңлҠ” ACK Slotмқ„ RecessiveлЎң м „мҶЎн•ҳкі , мҲҳмӢ м—җ м„ұкіөн•ң лӘЁл“  л…ёл“ңк°Җ мқҙ мң„м№ҳлҘј DominantлЎң лҚ®м–ҙм“ҙлӢӨ(ACK). м•„л¬ҙлҸ„ ACKлҘј ліҙлӮҙм§Җ м•Ҡмңјл©ҙ(лІ„мҠӨк°Җ Recessive мң м§Җ), мҶЎмӢ  л…ёл“ңлҠ” ACK ErrorлҘј мқёмӢқн•ңлӢӨ.

ACK Slot м „мҶЎ: 1 (Recessive)
лІ„мҠӨ мқҪкё°:     1 (Recessive) вҶҗ м•„л¬ҙлҸ„ ACK м•Ҳ н•Ё
вҶ’ ACK Error л°ңмғқ

5к°Җм§Җ м—җлҹ¬ л№„көҗ мҡ”м•Ҫ

м—җлҹ¬ мў…лҘҳкІҖм¶ң мЈјмІҙкІҖм¶ң м§Җм җмЈјмҡ” мӣҗмқё
Bit ErrorмҶЎмӢ  л…ёл“ңм „мҶЎ мӨ‘ лІ„мҠӨ лӘЁлӢҲн„°л§ҒлІ„мҠӨ кІ°н•Ё, м „кё°м Ғ к°„м„ӯ
Stuff ErrorмҲҳмӢ  л…ёл“ңлҚ°мқҙн„°/CRC н•„л“ңл…ёмқҙмҰҲ, лҸҷкё°нҷ” мҳӨлҘҳ
CRC ErrorмҲҳмӢ  л…ёл“ңCRC н•„л“ң л№„көҗлҚ°мқҙн„° мҶҗмғҒ
Form ErrorмҲҳмӢ  л…ёл“ңкө¬мЎ°м Ғ н•„л“ң мң„м№ҳн”„л Ҳмһ„ кө¬мЎ° мҶҗмғҒ
ACK ErrorмҶЎмӢ  л…ёл“ңACK SlotмҲҳмӢ  л…ёл“ң м—ҶмқҢ, лІ„мҠӨ лӢЁм„ 

мқҙ 5к°Җм§Җ мӨ‘ н•ҳлӮҳлқјлҸ„ мҳӨлҘҳлҘј к°җм§Җн•ҳл©ҙ, лӢӨмқҢ лӢЁкі„лҠ” мқҙлҘј лІ„мҠӨ м „мІҙм—җ м•ҢлҰ¬лҠ” кІғмқҙлӢӨ. мқҙл•Ң мӮ¬мҡ©лҗҳлҠ” кІғмқҙ м—җлҹ¬ н”„л Ҳмһ„мқҙлӢӨ.

2. м—җлҹ¬ н”„л Ҳмһ„

мҳӨлҘҳлҘј к°җм§Җн•ң л…ёл“ңлҠ” мҰүмӢң м—җлҹ¬ н”„л Ҳмһ„(Error Frame)мқ„ м „мҶЎн•ҙ лІ„мҠӨ м „мІҙм—җ мҳӨлҘҳ л°ңмғқмқ„ м•ҢлҰ°лӢӨ.

Active Error Flag (6 Dominant bits)

  • 6к°ңмқҳ м—°мҶҚ Dominant(0) л№„нҠё
  • мқҙ мһҗмІҙк°Җ Bit Stuffing к·ңм№ҷ мң„л°ҳмқҙлҜҖлЎң, лӢӨлҘё л…ёл“ңл“ӨлҸ„ Stuff ErrorлҘј к°җм§Җн•ҙ мһҗмӢ мқҳ м—җлҹ¬ н”Ңлһҳк·ёлҘј м „мҶЎн•Ё
  • кІ°кіјм ҒмңјлЎң лІ„мҠӨм—җ 6~12к°ңмқҳ м—°мҶҚ Dominant л№„нҠёк°Җ лӮҳнғҖлӮҳ лІ„мҠӨ м „мІҙк°Җ мҳӨлҘҳлҘј мқём§Җ

3. м—җлҹ¬ м№ҙмҡҙн„°

CAN м»ЁнҠёлЎӨлҹ¬лҠ” TEC(Transmit Error Counter)мҷҖ REC(Receive Error Counter) л‘җ к°ңмқҳ м№ҙмҡҙн„°лҘј мң м§Җн•ңлӢӨ. мқҙ м№ҙмҡҙн„°лЎң л…ёл“ңмқҳ м—җлҹ¬ мғҒнғңлҘј м¶”м Ғн•ңлӢӨ.

TEC (Transmit Error Counter) мҰқк°җ к·ңм№ҷ

мғҒнҷ©TEC ліҖнҷ”
м „мҶЎ мҳӨлҘҳ к°җм§Җ(м—җлҹ¬ н”„л Ҳмһ„ м „мҶЎ)+8
л©”мӢңм§Җ м„ұкіөм ҒмңјлЎң м „мҶЎ-1
Bus Off м§„мһ… мӢңTEC > 255 нҷ•м •

REC (Receive Error Counter) мҰқк°җ к·ңм№ҷ

мғҒнҷ©REC ліҖнҷ”
мҲҳмӢ  мҳӨлҘҳ к°җм§Җ (мІ« лІҲм§ё)+1
мҲҳмӢ  мҳӨлҘҳ к°җм§Җ (м—җлҹ¬ н”Ңлһҳк·ё нӣ„)+8
л©”мӢңм§Җ м„ұкіөм ҒмңјлЎң мҲҳмӢ -1
RECк°Җ 127 мҙҲкіј мғҒнғңм—җм„ң м„ұкіө мҲҳмӢ 119лЎң мһ¬м„Өм •

м—җлҹ¬ м№ҙмҡҙн„°мқҳ л№„лҢҖм№ӯм„ұ

м—җлҹ¬ м№ҙмҡҙн„°мқҳ л№„лҢҖм№ӯм„ұм—җ мЈјлӘ©н•ҳмһҗ. мҳӨлҘҳ мӢң +8, м„ұкіө мӢң -1лЎң м„Өкі„лҗң мқҙмң лҠ”, мҳӨлҘҳ л№ҲлҸ„к°Җ лӮ®м•„м•ј м •мғҒмқҙлқјлҠ” к°Җм • л•Ңл¬ёмқҙлӢӨ. к°„н—җм Ғ мҳӨлҘҳлҠ” м№ҙмҡҙн„°к°Җ мһҗм—°нһҲ нҡҢліөлҗҳм§Җл§Ң, м§ҖмҶҚм Ғ мҳӨлҘҳлҠ” л№ лҘҙкІҢ лҲ„м ҒлҗңлӢӨ.

к·ёл ҮлӢӨл©ҙ м№ҙмҡҙн„°к°Җ нҠ№м • мһ„кі„к°’м—җ лҸ„лӢ¬н•ҳл©ҙ м–ҙл–Ө мқјмқҙ лІҢм–ҙм§Ҳк№Ң? CANмқҖ м№ҙмҡҙн„° к°’м—җ л”°лқј л…ёл“ңмқҳ лҸҷмһ‘ мғҒнғңлҘј м„ё лӢЁкі„лЎң кө¬л¶„н•ңлӢӨ.

    BusOff --> ErrorActive: 128 Г— 11-recessive-bit мӢңнҖҖмҠӨ к°җм§Җ<br>(мһҗлҸҷ ліөкө¬)

Error Active (м—җлҹ¬ нҷңм„ұ)

  • мҙҲкё° мғҒнғңмқҙмһҗ м •мғҒ лҸҷмһ‘ мғҒнғң
  • TEC < 128, REC < 128
  • мҳӨлҘҳ к°җм§Җ мӢң Active Error Flag (6 Dominant bits) м „мҶЎ
  • лІ„мҠӨм—җ м Ғк·№м ҒмңјлЎң мҳӨлҘҳлҘј м•ҢлҰҙ мҲҳ мһҲмқҢ

Error Passive (м—җлҹ¬ нҢЁмӢңлёҢ)

  • TEC >= 128 лҳҗлҠ” REC >= 128 мӢң м§„мһ…
  • мҳӨлҘҳ к°җм§Җ мӢң Passive Error Flag (6 Recessive bits) м „мҶЎ
    • RecessiveмқҙлҜҖлЎң лӢӨлҘё л…ёл“ңмқҳ мӢ нҳёлҘј лҚ®м–ҙм“°м§Җ лӘ»н•Ё
    • лІ„мҠӨм—җ мҳӨлҘҳлҘј к°•м ңлЎң м•ҢлҰҙ мҲҳ м—ҶмқҢ
  • Interframe Spaceм—җм„ң м¶”к°Җ лҢҖкё° мӢңк°„(Suspend Transmission, 8 bits) л°ңмғқ

Bus Off (лІ„мҠӨ мҳӨн”„)

  • TEC > 255 мӢң м§„мһ…
  • л…ёл“ңк°Җ лІ„мҠӨм—җм„ң мҷ„м „нһҲ л¶„лҰ¬лҗЁ
  • м–ҙл– н•ң м „мҶЎлҸ„, мҲҳмӢ лҸ„ л¶Ҳк°ҖлҠҘ
  • н•ҳл“ңмӣЁм–ҙ лҰ¬м…ӢмқҙлӮҳ мһҗлҸҷ ліөкө¬ м Ҳм°ЁлҘј нҶөн•ҙм„ңл§Ң ліөк·Җ к°ҖлҠҘ

5. Bus Off ліөкө¬

Bus Off мғҒнғңм—җ м§„мһ…н•ң л…ёл“ңлҠ” л‘җ к°Җм§Җ л°©лІ•мңјлЎң ліөкө¬н•  мҲҳ мһҲлӢӨ.

мһҗлҸҷ ліөкө¬ (Auto Recovery)

CAN м»ЁнҠёлЎӨлҹ¬лҠ” лІ„мҠӨм—җм„ң 128лІҲмқҳ 11-recessive-bit мӢңнҖҖмҠӨлҘј к°җм§Җн•ҳл©ҙ мһҗлҸҷмңјлЎң Error Active мғҒнғңлЎң ліөк·Җн•ңлӢӨ.

ліөкө¬ мЎ°кұҙ:
лІ„мҠӨм—җм„ң 11к°ңмқҳ м—°мҶҚ Recessive(1) л№„нҠёлҘј 128лІҲ к°җм§Җ

128нҡҢ мҷ„лЈҢ мӢң вҶ’ TEC = 0, REC = 0, Error Active ліөк·Җ

11к°ң м—°мҶҚ RecessiveлҠ” мқён„°н”„л Ҳмһ„ мҠӨнҺҳмқҙмҠӨ(Interframe Space)мҷҖ мң нңҙ лІ„мҠӨ мғҒнғңм—җ н•ҙлӢ№н•ңлӢӨ. лІ„мҠӨк°Җ м •мғҒ лҸҷмһ‘ мӨ‘мқҙл©ҙ мқҙ мӢңнҖҖмҠӨк°Җ мһҗм—°нһҲ л°ңмғқн•ңлӢӨ.

мҲҳлҸҷ ліөкө¬ (Manual Recovery)

мқјл¶Җ CAN м»ЁнҠёлЎӨлҹ¬лҠ” мҶҢн”„нҠёмӣЁм–ҙм—җм„ң Bus Off мһҗлҸҷ ліөкө¬лҘј л№„нҷңм„ұнҷ”н•  мҲҳ мһҲлӢӨ. мқҙ кІҪмҡ° м• н”ҢлҰ¬мјҖмқҙм…ҳ мҪ”л“ңк°Җ лӘ…мӢңм ҒмңјлЎң ліөкө¬лҘј нҠёлҰ¬кұ°н•ҙм•ј н•ңлӢӨ.

can_set_mode(CAN1, CAN_MODE_NORMAL); /* Re-enter normal mode */

н•өмӢ¬ м •лҰ¬

  • CANмқҖ Bit, Stuff, CRC, Form, ACK 5к°Җм§Җ лҸ…лҰҪ л©”м»ӨлӢҲмҰҳмңјлЎң мҳӨлҘҳлҘј кІҖм¶ңн•ҳл©°, н•ҳлӮҳлқјлҸ„ к°җм§Җлҗҳл©ҙ м—җлҹ¬ н”„л Ҳмһ„мқ„ м „мҶЎн•ңлӢӨ.
  • м—җлҹ¬ н”„л Ҳмһ„мқҖ 6к°ңмқҳ Dominant(Active) лҳҗлҠ” 6к°ңмқҳ Recessive(Passive) л№„нҠёлЎң кө¬м„ұлҗң м—җлҹ¬ н”Ңлһҳк·ёмҷҖ 8к°ңмқҳ Recessive DelimiterлЎң мқҙлӨ„м§„лӢӨ.
  • TECлҠ” м „мҶЎ мҳӨлҘҳ мӢң +8, м„ұкіө мӢң -1. RECлҠ” мҲҳмӢ  мҳӨлҘҳ мӢң +8, м„ұкіө мӢң -1лЎң ліҖн•ңлӢӨ.
  • м—җлҹ¬ мғҒнғңлҠ” Error Active вҶ’ Error Passive вҶ’ Bus Off мҲңмңјлЎң м•…нҷ”лҗҳл©°, к°Ғ мһ„кі„к°’мқҖ 128, 255мқҙлӢӨ.
