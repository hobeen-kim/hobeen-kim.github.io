데이터베이스

관계형 데이터베이스를 중심으로, SQL 기초부터 내부 구조, 튜닝, 분산 데이터베이스까지 체계적으로 학습한다. 정보처리기사 출제 범위를 모두 포함하며, 실무 관점의 튜닝과 운영까지 다룬다.

목차

기초

  1. 데이터베이스란 — 정의, DBMS, 스키마, 데이터 독립성
  2. 관계형 모델 — 테이블/키/관계, ERD, 관계 대수

SQL

  1. DDL과 DCL — CREATE/ALTER/DROP, GRANT/REVOKE
  2. SQL 기초 — SELECT, WHERE, ORDER BY, DML
  3. JOIN과 서브쿼리 — INNER/LEFT/RIGHT JOIN, 서브쿼리
  4. 집계와 윈도우 함수 — GROUP BY, HAVING, ROW_NUMBER
  5. 데이터베이스 객체 — VIEW, 저장 프로시저, 트리거, 시퀀스

설계

  1. 정규화 — 이상 현상, 함수적 종속, 1NF~BCNF, 반정규화
  2. 데이터 모델링 — 설계 4단계, 논리/물리 모델, ERD 실무

내부 구조

  1. 스토리지 엔진 — 페이지, B-Tree, 해시 인덱스, 클러스터드
  2. 트랜잭션과 동시성 — ACID, 격리 수준, MVCC, 병행 제어, 회복
  3. 실행 계획 — EXPLAIN, 옵티마이저, 스캔 방식

튜닝

  1. 인덱스 전략 — 복합 인덱스, 커버링 인덱스, 설계 원칙
  2. 쿼리 튜닝 — 슬로우 쿼리, 안티패턴, 개선 사례

운영과 확장

  1. 운영과 보안 — 커넥션 풀, 파티셔닝, 접근 제어
  2. 분산 데이터베이스 — 2PC, CAP 정리, 샤딩, 복제
  3. 데이터 웨어하우스 — OLTP vs OLAP, Star 스키마, ETL
